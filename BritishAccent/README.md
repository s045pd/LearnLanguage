## 【节奏】

### [1.重音等时](https://en.wikipedia.org/wiki/Isochrony#Syllable_timing)(**stress timing**)

> 句子中重音间的时间间隔近乎相同

而中文则是音节等时(**syllable timing**)，由此一字一停造就了中式英语，需优先纠正。


尝试以相同的停顿时长读以下句子:

|       1秒       |       1秒       |    1秒    |
| :-------------: | :-------------: | :-------: |
|      dogs       |      chase      |   cats    |
|    the dogs     |      chase      | the casts |
| the clever dogs | will be chasing | the cats  |
